# MPTCP-Video-Streaming
A flask Server to implement Video Streaming using MPTCP

## Guides

### For setting up MPTCP-
  Refer https://multipath-tcp.org/pmwiki.php/Users/HowToInstallMPTCP

  
   ### To set the environment:
  ```
  $ pipenv shell 
  $ pipenv install
  ```
 ### To run the server
  ```
  python server.py
  ```
